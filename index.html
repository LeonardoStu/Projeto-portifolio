Habilidades

Javascript

JavaScript é uma linguagem de programação de alto nível, leve e orientada a eventos, usada principalmente para criar interatividade em páginas web. Ele roda no navegador e no servidor (com Node.js) e suporta paradigmas funcionais, orientados a objetos e assíncronos.

Typescript

TypeScript é um superconjunto do JavaScript que adiciona tipagem estática e recursos avançados, como interfaces e enums. Ele ajuda a evitar erros e melhora a produtividade, sendo amplamente usado em projetos grandes e escaláveis.

Sass

Sass (Syntactically Awesome Stylesheets) é um pré-processador de CSS que adiciona recursos como variáveis, aninhamento, mixins e herança, tornando o código mais organizado e reutilizável. Ele é compilado para CSS antes de ser usado no navegador.

React

React é uma biblioteca JavaScript para criar interfaces de usuário, focada em componentes reutilizáveis e no conceito de virtual DOM para otimizar renderizações. É amplamente usada no desenvolvimento web moderno, especialmente em aplicações SPA (Single Page Applications).

Next

Next.js é um framework React que facilita a criação de aplicações web otimizadas, oferecendo renderização híbrida (SSR e SSG), roteamento automático, API Routes e suporte a SEO. É ideal para projetos escaláveis e de alto desempenho.

MongoDB

MongoDB é um banco de dados NoSQL orientado a documentos, que armazena dados em formato JSON/BSON. Ele é escalável, flexível e ideal para aplicações que exigem alta performance e grande volume de dados.

Postgres

PostgreSQL é um banco de dados relacional open-source, conhecido por sua robustez, conformidade com ACID e suporte a SQL avançado. Ele é altamente escalável e usado em aplicações que exigem integridade e complexidade nos dados.

Git e Github

Git é um sistema de controle de versão distribuído que permite rastrear mudanças no código e colaborar em projetos. GitHub é uma plataforma baseada em Git que hospeda repositórios, facilita o versionamento e a colaboração entre desenvolvedores, além de oferecer CI/CD e outras ferramentas.

HTML

HTML (HyperText Markup Language) é a linguagem de marcação usada para estruturar páginas web. Ele define elementos como textos, imagens, links e formulários, sendo a base do desenvolvimento web junto com CSS e JavaScript.

CSS

CSS (Cascading Style Sheets) é a linguagem usada para estilizar páginas web, definindo cores, layouts, fontes e animações. Ele permite separar a estrutura (HTML) da apresentação, tornando o design mais flexível e responsivo.

Node

Node.js é um ambiente de execução JavaScript do lado do servidor, baseado no motor V8 do Chrome. Ele permite criar aplicações rápidas e escaláveis, especialmente para servidores e APIs, usando JavaScript.
